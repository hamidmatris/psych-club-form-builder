== Description ==

**Psych Club Form Builder** یک افزونه فرم‌ساز پیشرفته برای وردپرس است که ویژگی‌های زیر را ارائه می‌دهد:

### ویژگی‌های اصلی:
- 🏗️ فرم‌ساز بصری با قابلیت کشیدن و رها کردن
- 📋 پشتیبانی از انواع فیلدها (متنی، ایمیل، عددی، تاریخ، فایل و...)
- 🛡️ سیستم ضد اسپم پیشرفته
- 📧 اعلان‌های ایمیلی
- 📊 مدیریت و آنالیز نتایج
- 📱 طراحی ریسپانسیو
- 🔒 امنیت بالا با nonce verification

### انواع فیلدهای پشتیبانی شده:
- متن تک خطی
- متن چند خطی
- ایمیل
- عدد
- تلفن
- تاریخ
- آدرس وب
- چک‌باکس
- دکمه رادیویی
- لیست انتخابی
- آپلود فایل

== Installation ==

1. فایل زیپ افزونه را دانلود کنید
2. به پیشخوان وردپرس بروید و به بخش «افزونه‌ها > افزودن» بروید
3. افزونه را آپلود و نصب کنید
4. افزونه را فعال کنید
5. به منوی «فرم‌ساز روانشناسی» در پیشخوان بروید

== Frequently Asked Questions ==

= چگونه یک فرم جدید ایجاد کنم؟ =
به منوی «فرم‌ساز روانشناسی» بروید و روی «ایجاد فرم جدید» کلیک کنید.

= چگونه فرم را در صفحه قرار دهم؟ =
از شورتکد `[pcfb_form id="1"]` استفاده کنید که ID فرم را جایگزین عدد 1 کنید.

= چگونه از اسپم جلوگیری کنم؟ =
در تنظیمات افزونه، بخش «حفاظت ضد اسپم» را فعال کنید.
